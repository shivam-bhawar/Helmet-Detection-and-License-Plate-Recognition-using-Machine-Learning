# Helmet-Detection-and-License-Plate-Recognition-using-Machine-Learning
BE-Final Year Project

This System is used to detect Motorcycle Riders without Helmets. Then this System can detect and extract corresponding motorcycle's License Plate Number and save it into the System Database so that the Rider can be charged with fine. We have used YOLOv4 i.e. You Only Look Only Once Version 4 Algorithm to detect the Helmet as object in Input Video and we have used OCR i.e. Optical Character Recognition Algorithm to detect and extract the corresponding License Plate Number.

Technologies Used:
1. Python
2. YOLOv4 (You Only Look Once Version 4)
3. OCR (Optical Character Recognition)
4. OpenCV
5. Keras
6. Pillow
